== Description ==

Caches Gravatars to your host and speed up your site.


== Installation ==

1. Upload `nix-gravatar-cache` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy
